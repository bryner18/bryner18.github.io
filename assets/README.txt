assets/
=======
fonts/  — locally bundled web fonts so typography is identical on
          desktop, Android and iOS (no OS font substitution):

  comic-neue-*.woff2      Comic Neue      (Comic Sans MS stand-in, OFL)
  anton-400.woff2         Anton           (Impact stand-in, OFL)
  courier-prime-*.woff2   Courier Prime   (Courier New stand-in, OFL)
  tinos-400.woff2         Tinos           (Times New Roman metric twin, Apache 2.0)
  ms_sans_serif*.woff2/.woff  Pixelated MS Sans Serif (from 98.css, MIT)

Every "GIF" on this site is CSS animation + emoji, so no image assets
are needed and the whole thing works offline — very 1998 of us.
